# Eye-Disease-Prediction
OCT-based retinal disease prediction web app using machine learning, Streamlit, Firebase authentication, and email notification.
# 👁️ Eye Disease Prediction using OCT

This project is a machine learning-based web application that predicts retinal diseases from OCT images.

## 🚀 Features
- User Login & Signup (Firebase)
- OCT Image Upload
- Disease Prediction (8 Classes)
- Confidence Percentage
- Email Notification with Recommendations

## 🧠 Technologies Used
- Python
- TensorFlow / CNN (MobileNetV3)
- Streamlit
- Firebase Authentication
- SMTP (Email)

## 📌 Diseases Detected
CNV, DME, DRUSEN, NORMAL, AMD, CSR, DR, MH

## ▶️ Run Locally
```bash
streamlit run app.py
